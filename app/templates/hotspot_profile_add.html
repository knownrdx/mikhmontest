How It Works

When you create a profile, MikroMan will:

  • Create the profile on your MikroTik router
  • Set up the on-login script for auto-activation & expiry
  • Create a scheduler for auto-removal of expired users
  • Store pricing metadata for sales reporting
Tips
  • Validity: Use d=days, h=hours, m=minutes (e.g. 30d, 24h, 1d12h)
  • Rate Limit: Format is upload/download (e.g. 2M/5M)
  • Remove & Record: Best option — auto-deletes expired users and logs sales
  • MAC Lock: Prevents credential sharing — only first device can use the voucher
