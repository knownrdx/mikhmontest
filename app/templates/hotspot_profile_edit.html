What Gets Updated

Saving will update on the MikroTik router:

  • Profile settings (pool, queue, rate, shared)
  • on-login script with new :put metadata & validity
  • Per-profile expiry scheduler
  • Global expiry scheduler (backward compat)
Important Notes
  • Changing validity only affects new users — existing users keep their original expiry
  • Renaming a profile won't rename already-assigned users on RouterOS
  • Rate limit changes apply immediately to all users of this profile
